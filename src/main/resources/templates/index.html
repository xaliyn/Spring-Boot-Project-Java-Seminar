Welcome to TownsApp

TownsApp is a Java Spring Boot web application developed for the Java Applications Seminar course at John von Neumann University. The project demonstrates how to create a modern web application using Spring Boot, Thymeleaf, MySQL, and Chart.js for data visualization.

Users can view data about Hungarian towns and counties, access statistics, and visualize population trends. Future seminar tasks will extend this project with authentication, database CRUD operations, and RESTful APIs.
