Project Description

TownsApp is a Spring Boot web application created for the Java Applications Seminar at John von Neumann University (GAMF Faculty). It demonstrates real-world backend development using:

  • Spring Boot (MVC, Security, Data JPA)
  • Thymeleaf templating
  • MySQL database
  • REST API endpoints
  • HTML5UP Photon theme

The application manages Hungarian towns, counties, and population data. It includes:

  • User registration & login
  • Protected pages
  • CRUD operations for entities
  • Data listing and statistics
